# My Enterprise Cybersecurity Home Lab

## 🌐 Network Architecture
To safely practice security testing, I built an isolated virtual network environment.
* **Hypervisor:** VirtualBox & UTM
* **Network Type:** Host-Only (isolated from the physical home network and the internet)

## 🖥️ System Inventory
* **Attacking Machine:** Parrot OS (Fully updated with offensive security tools)
* **Vulnerable Endpoint:** Metasploitable 2 (Linux-based server hosting bWAPP and vulnerable network daemons)
